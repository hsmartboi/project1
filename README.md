## 📚 핵심 개념

### 1. **LIF (Leaky Integrate-and-Fire) 뉴런 모델**

생물학적 뉴런을 수학적으로 모델링한 가장 기본적인 형태입니다.

**미분방정식**:
```
τ_m * dV/dt = -(V - V_rest) + R*I
```

**구성 요소**:
- **막전위 (V)**: 뉴런의 전기적 상태
- **임계값 (θ)**: 스파이크 발생 기준 전위
- **불응기 (τ_ref)**: 스파이크 후 입력 불감응 시간
- **시정수 (τ_m)**: 신호 감쇠 속도

**동작 원리**:
1. 입력 신호 → 막전위 상승
2. 막전위 ≥ 임계값 → **스파이크 발생**
3. 스파이크 후 → 막전위 리셋 및 불응기 진입
4. 불응기 중 → 입력 신호 무시

### 2. **STDP (Spike-Timing-Dependent Plasticity)**

스파이크 타이밍에 따른 시냅스 가중치 변화 규칙입니다.

**학습 규칙**:
- **Post-Pre (후시냅스가 먼저 발화)**: Δw = A_+ * exp(-Δt/τ) → **LTP (가중치 증가)**
- **Pre-Post (전시냅스가 먼저 발화)**: Δw = -A_- * exp(Δt/τ) → **LTD (가중치 감소)**

**생물학적 의미**:
- 인과관계 있는 뉴런 간 연결 강화
- 뇌의 자기주도적 학습 메커니즘

### 3. **Spike Encoding (입력 인코딩)**

실수 데이터를 스파이크 신호로 변환합니다.

**Rate Coding**: 데이터 크기 ∝ 스파이크 빈도

## 📊 시뮬레이션 설명

### **시뮬레이션 1: 기본 피드포워드 신경망**

**구조**: 입력층(2) → 은닉층(3) → 출력층(1)

**목표**: 시간에 따라 변하는 입력 패턴을 신경망이 처리하는 과정 관찰

**입력 패턴**:
- 처음 50 스텝: [0.8, 0.2]
- 나머지 50 스텝: [0.2, 0.8]

**예상 결과**:
- 입력 변화에 따른 뉴런 활성화 변화
- 신경망의 정보 처리 과정 시각화

---

### **시뮬레이션 2: STDP 학습**

**목표**: Pre-Post 스파이크 타이밍에 따른 가중치 변화 관찰

**구조**: 뉴런 0 → 뉴런 1, 2 (2개 시냅스)

**입력 패턴**:
- **시냅스 1**: Pre(t=10,40,70) → Post(t=15,45,65)
  - Δt = +5ms → **LTP (가중치 증가)** ✓
  
- **시냅스 2**: Pre(t=10,40,70) → Post(t=5,35,75)
  - Δt = -5ms, +5ms → **LTD + LTP** (변화 없음)

**예상 결과**:
- 시냅스 1의 가중치 증가
- 시냅스 2의 가중치 변화 미미
- 인과관계 기반 학습 증명

---

### **시뮬레이션 3: 무작위 신경망**

**구조**: 5개 뉴런의 무작위 연결 (40% 연결 확률)

**목표**: 복잡한 네트워크에서의 동적 활성화 패턴 분석

**입력**: 포아송 프로세스 기반 입력 (30% 발화율)

**예상 결과**:
- 네트워크의 동적 응답
- 발화율 분석
- STDP에 의한 가중치 진화

## 📈 시각화 출력

각 시뮬레이션 후 다음 3가지 그래프가 표시됩니다:

### 1. **Spike Raster Plot**
```
시간(x축) vs 뉴런 인덱스(y축)
- 빨간 선: 스파이크 발생 시점
- 각 뉴런의 발화 패턴 한눈에 파악 가능
```

### 2. **Membrane Potential**
```
시간(x축) vs 막전위(y축)
- 각 뉴런의 막전위 변화 추적
- 임계값(1.0) 도달 시 스파이크 발생
```

### 3. **Synaptic Weights**
```
STDP 업데이트 단계(x축) vs 시냅스 가중치(y축)
- STDP 학습에 따른 가중치 진화
- 시냅스별 학습 효율성 비교
```
